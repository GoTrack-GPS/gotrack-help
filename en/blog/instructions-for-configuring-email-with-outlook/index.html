Instructions for configuring email with Outlook

Posted June 25, 2022 by NamNH

Log in #

To log in to Outlook, do the following:

Step 1 #

  • Open Outlook on your computer ( if your computer does not have Outlook software installed, you can refer to it here ).

Step 2 #

2.1. Open the app #

  • When successfully opening the application, it will display as shown below:

2.2. Add account #

2.2.1. Add #

  • Click on File -> Info -> Add Account
2.2.2. Log in #

  • Enter your Email.
  • Tick on Let me set up my account manually to install configuration manually.
  • Click Connect to continue.
2.2.3. Configuration #

  • Select POP will display as shown below:

    • Incoming mail: Incoming mail server (POP.GOTRACK.VN).
      • Server : Your Mail Incoming Email Server (for example incoming mail server The email of tk tester@gotrack.vn is mail.gotrack.vn).
      • Port : POP3 Port

    • Outgoing mail: Outgoing mail server (SMTP.GOTRACK.VN).
      • Server : Outgoing Mail Server Your Email (for example outgoing mail server tk tester@gotrack.vn’s email is mail.gotrack.vn).
      • Port : SMTP Port

  • Click Next enter Password of Email and then click Connect
  • After successful login click Done to finish.

If you want to add more Emails, do the same from Step 2.2.1
