Installation and User Manual MDVR (SM400) #

Step 1 : Hardware preparation #

  • SM400 Basic or Livestream recorder.
  • Equipment conforms to QCVN31 S200 or S400 standards.
  • Camera.
  • 4G sim for SM400 livestream recorder (if available).
    (No need for SM400 base recorder).
  • 4G sim for QCVN31 standard equipment.
  • Power cord connects standard equipment and recorder.
  • And other accessories included: RFID card, SD card, GPS antenna, GSM (if any).

  • Prepare tools, tools:
    • TFT LCD HD Screen Integrated Car CCTV (7inch): For convenience when updating firmware/configuration or when handling errors.
    • USB at least 1G : Save firmware and configuration files

  • Devices to install SM400 Basic as shown below:

  • Devices to install SM400 Livestream as shown below:

Step 2 : Wiring connection #

Applies to both SM400 Basic and Livestream recorders.

  • Plug the cord with a QCVN-compliant device connector to the SM400 recorder.

  • Plug the camera into the recorder in order from AV1 -> AV4.
  • Insert SD memory card into SM400 recorder.
  • Insert 4G sim (only for SM400 livestream) into SM400 recorder.
  • Connect the SM400 recorder to the monitor for configuration.
  • Power on the SM400 recorder.

Step 3 : Camera mounting position #

❗
It is recommended to install the camera position according to ND10

Instructions for installing the camera ensure that the entire working process of the driver is observed. #

  • The position of installing the camera eye must ensure that the image is observed when the driver works, the image must ensure that the driver is visible from the waist up and the entire steering wheel controls the vehicle.
  • The optimal camera installation position is in front of the driver, in a space that ensures an angle of α ≤ 60 degrees (as shown in the figure below).
    • Depending on the type of camera eye, the mounting unit can be mounted above the driver’s head (between the steering wheel and the front of the vehicle).

Step 4 : Recorder configuration #

  • (Only applicable for SM400 Livestream)

  • GoTrack camera server IP settings

    • 171.244.003.120

  • Video settings: basic settings, main stream settings, sub stream settings, time settings, memory management, OSD settings

  • Basic setting: basic recording parameter setting

    • Video format: PAL, NTSC, press 【OK】 key to choose from drop-down menu.
    • Video mode: start-up video, timer video, alarm video, start-up recording: start recording automatically when device is on recording Timed recording: start recording within a timed recording interval and not recorded at other times. Alarm recording: the device starts recording when there is an alarm.
    • Audio format: G726-M40K / G726-M32K / G726M40K / G726M32K / G711A.
    • Sound gain: 0-14 levels optional, volume adjustment.
    • Pre-alarm recording: record a period of time before the alarm.
    • Alarm delay: the recording will continue after dismissing the alarm and the recording time is the set time.
    • Alarm protection: In the video file list interface, select the video file to lock, it will not be automatically overwritten during alarm protection.
    • Camera type: Select according to the camera’s resolution.

  • Main stream setting: setting recording parameters for each channel

    • Channel name: preview and playback screen will be displayed.
    • Main Recording: On: means the channel has the recording function on.
      ❗
      OFF: means the channel is not recording.
    • Resolution: 720P, D1, HD1 and CIF can be selected. For example, in PAL system: 720P resolution is 1280 * 720, D1 resolution is 704 * 576, HD1 resolution is 704 * 288, CIF is 352 * 288.
    • Frame rate: indicates the number of video frames per second, PAL system is 1~25fps, NTSC system is 1~30fps.
    • Image quality, image clarity, 1 ~ 8 levels, level 1 picture quality is the highest.
    • Recording: On/off, indicates whether to record audio while recording video or not.

  • Sub stream setting: setting parameters of 4G network transmission video stream

    • Sub video: on / off, whether to enable the video sub stream function or not.
    • Resolution: CIF/HD1/D1 optional.
    • Frame rate/image quality/recording: same as main stream setting interface parameters.

  • Time setting: set time recording interval

    • You can set up to 4 recording intervals per day.You can set up to 4 recording intervals per day.
    • Date: The specified period is valid for 7 days from Monday to Sunday.
    • Time zone: the start time of the time zone cannot be greater than the end time.

  • Storage management:

    • Purpose: Main video / secondary video option, select video stream.
    • Priority: high/medium/low.

  • OSD setting: overlay information and display position of recorded video

    • Stacked items: time / license plate / phone number / channel number / GPS information.
    • Trigger: open/close.
    • X/Y coordinates: the position where the information is displayed on the screen that overlaps.
