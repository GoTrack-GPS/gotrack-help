Hướng dẫn cấu hình email với Outlook

Xuất bản ngày April 8, 2022 bởi NamNH

Đăng nhập #

Để đăng nhập vào Outlook cần thực hiện như sau :

Bước 1 #

  • Mở Outlook trên máy tính của bạn ( nếu máy tính của bạn chưa cài đặt phần mềm Outlook thì của thể tham khảo tại đây ).

Bước 2 #

2.1. Mở ứng dụng #

  • Khi mở ứng dụng thành công sẽ hiển thị như hình dưới đây :

  • Nếu hiển thị đăng nhập như dưới đây thì tiếp tục thực hiện bước 2.2.2

2.2. Thêm tài khoản. #

2.2.1. Thêm #

  • Click vào File -> Info -> Add Account
2.2.2. Đăng nhập #

  • Nhập Email của bạn.
  • Tick vào Let me set up my account manually để cài đặt cấu hình thủ công.
  • Ấn Connect để tiếp tục.
2.2.3. Cấu hình #

  • Chọn POP sẽ hiển thị như hình dưới đây:

    • Incoming mail: Máy chủ thư đến (POP.GOTRACK.VN).
      • Server : Server máy chủ thư đến Email của bạn (ví dụ server máy chủ thư đến Email của tk tester@gotrack.vn là mail.gotrack.vn).
      • Port : Cổng POP3 Port

    • Outgoing mail: Máy chủ thư đi (SMTP.GOTRACK.VN).
      • Server : Server máy chủ thư đi Email của bạn (ví dụ server máy chủ thư đi Email của tk tester@gotrack.vn là mail.gotrack.vn).
      • Port : Cổng SMTP Port

  • Click Next nhập Mật khẩu của Email rồi click Connect
  • Sau khi đăng nhập thành công click Done để kết thúc.

Nếu muốn thêm nhiều Email khác thì làm tương tự từ Bước 2.2.1
