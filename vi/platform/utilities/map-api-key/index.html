Map Api Key #

  • Map Api Key: Là tiện ích cung cấp dữ liệu địa chỉ cho trang bản đồ của tài khoản.

  • Để xem MAP API trong tài khoản làm như sau:

    • Click vào Tiện ích > Map Api Key

1. Tạo Map Api Key #

  • Bước 1: Click Tạo Map API key.

  • Bước 2: Xuất hiện hộp thoại:
    • Tên: Đặt tên cho MAP API.
    • Loại:
      • Basic, GoTrack-VN, GoTrack-VN Beta: Không cần mã.
      • Google, Mapbox, Goong: Cần nhập mã là API KEY của mỗi loại.
    • Mô tả: Điền mô tả chi tiết.
    • Trạng thái: Click Bật / Tắt trạng thái kích hoạt của MAP API KEY.

  • Bước 3: Click Thêm để thêm mới API.

2. Tìm kiếm #

  • Bước 1: Có thể tìm kiếm theo tiêu chí sau:
    • Tên: Nhập tên Map Api Key, nhập 1 hoặc nhiều ký tự để tìm kiếm.
    • Loại: Chọn loại dữ liệu bản đồ.
    • Mã: Nhập mã cần tìm, nhập 1 hoặc nhiều ký tự để tìm kiếm.
    • Trạng thái: Chọn trạng Kích hoạt / Không kích hoạt.
    • Mô tả: Nhập mô tả muốn tìm, nhập 1 hoặc nhiều ký tự để tìm kiếm.
    • Tài khoản con: Tick vào nếu muốn tìm trên tài khoản con.

  • Bước 2: Click vào Tìm kiếm để xem danh sách API hoặc Đặt lại để đặt lại về mặc định.

3. Sửa #

  • Bước 1: Click vào Sửa

  • Bước 2:
    • Tên: Sửa tên cho MAP API.
    • Loại:
      • Basic, GoTrack-VN, GoTrack-VN Beta: Không cần mã.
      • Google, Mapbox, Goong: Cần nhập mã là API KEY của mỗi loại.
    • Mô tả: Điền mô tả chi tiết.
    • Trạng thái: Click Bật / Tắt trạng thái kích hoạt của MAP API KEY.

  • Bước 3: Click Lưu thay đổi để hoàn thành thao tác.

4. Xóa #

  • Bước 1: Click vào > Xóa.

  • Bước 2: Click OK nếu bạn đồng ý xóa.

5. Xuất báo cáo #

  • Khi tìm kiếm ra kết quả, click vào Tải xuống để xuất dữ liệu:
    • XLSX: Tải xuống dạng Excel của trang hiện tại.
    • PDF: Tải xuống dạng PDF của trang hiện tại.
    • Tải xuống tất cả XLSX: Tải xuống dạng Excel tất cả các trang.
    • Tải xuống tất cả PDF: Tải xuống tất cả PDF tất cả các trang.

  • Click vào để làm mới danh sách.
  • Click vào để chỉnh sửa chiều cao của hàng theo mức Nhỏ -> Lớn.
    • Nhỏ : Chiều cao của từng hàng sẽ nhỏ lại.
    • Trung bình : Chiều cao của hàng sẽ rộng ra cách chữ 1 chút.
    • Lớn : Chiều cao của hàng sẽ rộng hơn loại Trung bình thêm 1 chút.
  • Click vào để hiển thị dữ liệu dạng bảng.
    • Bật / tắt hiển thị bảng / cột theo ý bạn muốn.
    • Đặt lại sẽ chọn lại về mặc định hiển thị tất cả.
